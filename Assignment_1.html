Basics of Web Technology

This static web page is created using only HTML and CSS to explain key concepts of web technology in a clean and simple format.

What is Web Technology?

Web technology refers to the tools and languages used to build and manage websites and web applications. These include:

  • HTML (HyperText Markup Language): Structure of web pages
  • CSS (Cascading Style Sheets): Styling and layout
  • JavaScript: Interactivity and dynamic behavior

How the Web Works

  • Websites are hosted on web servers and accessed using web browsers (like Chrome, Firefox).
  • Users request web pages by entering a URL (Uniform Resource Locator).
  • Communication happens using HTTP or HTTPS protocols.

Frontend vs Backend

  • Frontend deals with the part of a website users interact with (HTML, CSS, JavaScript).
  • Backend handles database interaction, authentication, and server logic (PHP, Node.js, Python, etc.).

Tools and Technologies

  • Text Editors: VS Code, Sublime Text
  • Version Control: Git, GitHub
  • Frameworks: Bootstrap (CSS), React.js (JavaScript)
